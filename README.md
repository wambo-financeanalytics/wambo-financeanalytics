## 💼 Projects

### Data Jobs Analysis
This project explores trends in the data job market, focusing on roles such as Data Analysts and Data Scientists. It examines employment patterns, salary levels, skills demand, and working conditions in the analytics industry.

The analysis was conducted using Power BI and includes interactive visualizations that present insights across job roles, salary distribution, geographic trends, and skill requirements.

Key focus areas included:
- Data job roles and salary trends  
- Education and qualification requirements  
- Remote vs on-site work distribution  
- Geographic distribution of data jobs  
- Skills demand in the data industry  

The project provides insights into how the data job market is structured and highlights key opportunities and expectations for individuals pursuing careers in analytics.
